게임시작 - SvSquare 페이지 연결

시작마을 - 대장간, 광장, 상점, 검문소
 - 대장간 : 무기 강화 가능
 - 광장 : 시작지점, 마을 내 사이트 간 연결
 - 상점 : 물약 등을 구입 가능
 - 검문소 : 숲으로 나갈 수 있음

숲 - 마을 앞, 깊은 숲 1, 깊은 숲 2, 옆마을 앞
 - 마을 앞 : 토끼, 뱀, 사슴 사냥 가능
 - 깊은 숲 1 : 여우, 곰, 늑대 출현
 - 깊은 숲 2 : 고블린, 오크 출현
 - 옆마을 앞 : 도적 출현 

옆마을 - 점령당한 검문소, 폐허가 된 광장, 병영, 점령당한 시청
 - 점령당한 검문소 : 산적 경비병 출현
 - 폐허가 된 광장 :  산적, 산적떼, 감시병 산적 출현
 - 병영 : 산적떼, 강화 산적떼 출현
 - 점령당한 시청 : 산적떼, 산적떼 두목 출현

v0: 시작 시 광장에서 시작, 상점이랑 대장간은 일단 작업 없음, 아이템 따로 없음, "한번 더 돌기" 기능 추가, Battle 페이지 추가,
광장 - 마을 내 사이트 이동 버튼 (일단 상점이랑 대장간 이동 버튼은 막아둠)
상점 - 광장 이동 버튼, 상점 주인 대사 추가
대장간 - 광장 이동 버튼, 대장간 주인 대사 추가
검문소 - 밖으로 나가기 버튼, 광장 이동 버튼, 경비병 대사 추가
마을 앞 - 마을 앞 메인 페이지 ~ battle 페이지 연결, 한번 더 돌기 버튼, 깊은 숲 1 이동 버튼(일단 막아둠), 검문소 이동 버튼

페이지 작업
main - 각 몬스터 및 주인공 info
SvSquare - 게임 시작 시 연결되는 페이지, 각 사이트 간 연결 버튼(상점이랑 대장간은 연결 없이 "아직은 갈 수 없다" 텍스트만 띄움)  
gate - 경비병 대사 출력, 광장 가는 버튼, 마을 앞 가는 버튼
SvFront1 - "몬스터를 만났다!" 텍스트 출력, 교전 버튼(클릭 시 battle 페이지로 넘어감) 
SvFront2 - "몬스터를 해치웠다!" 텍스트 출력, 한번 더 돌기 버튼(클릭 시 SvFront1 페이지 연결), 깊은 숲 1 이동 버튼(일단 막아둠), 검문소 이동 버튼
battle - 교전, 해당 페이지에 따라 등장 몬스터 구분 - main에 각 맵에 등장할 몬스터 목록 제작, 목록 내에서 랜덤으로 몬스터를 꺼내와서 battle 알고리즘 돌림, "이전 페이지에서 해당 배틀 맵 숫자 같이 보냄 ->
 배틀 페이지에서 main으로 이동 -> main에서 해당 번호 목록 가져옴 -> 배틀 페이지에서 목록 내 몬스터 랜덤으로 가져와서 시작
